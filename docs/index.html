🚀 Início Rápido

1. Clonar o repositório na versão correta

git clone https://github.com/SEU_USUARIO/edge-ml-vibration-classifier.git
cd edge-ml-vibration-classifier
git checkout v2.0.0   # versão estável mais recente

2. Treinar e exportar o modelo

pip install tensorflow scikit-learn numpy
python model/train.py
python model/quantize.py
# → gera: model/artifacts/vibration_classifier_int8.tflite

3. Flash no ESP32 (ESP-IDF)

cp model/artifacts/vibration_classifier_int8.tflite firmware/
idf.py build flash monitor

🏗️ Arquitetura do Pipeline

Sensor: ADXL345 via SPI @ 1 kHz → Buffer: 128 amostras (janela deslizante) → Inferência: TFLite Micro INT8 no ESP32 (~18 ms) → Saída: 0=Normal · 1=Falha Iminente · 2=Falha → Alerta: MQTT → Dashboard
